# OpsX Explorer

A minimal VS Code extension that registers an `OpsX Explore` command in the Command Palette.

## Setup

1. Open `c:\Users\USER\Desktop\marble-app\opsx-extension` in VS Code.
2. Run `npm install`.
3. Press `F5` to launch the Extension Development Host.
4. Open the Command Palette and run `OpsX Explore`.

## What it does

- Registers the `opsxExplorer.explore` command.
- Shows a simple action menu in VS Code.
- Can open `requirement.md` in the workspace.
